2. InstallationÂ¶

prerequisites

NumPy, SciPy

Download either NREL_CSM.py-0.1.0.tar.gz or NREL_CSM.py-0.1.0.zip, and uncompress/unpack it.

Install NREL_CSM with the following command.

$ python setup.py install

To check if installation was successful import the model or run the example.

$ python
> from NREL_CSM.csm import csm
> from NREL_CSM.csm import example
> example()
